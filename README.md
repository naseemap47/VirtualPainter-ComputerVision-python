## Description
Creating a Virtual Painter using OpenCV and Mediapipe in python

## Requirements
#### Programming Language
* Python3 (pyCharm)
#### Libraries
* OpenCV
* Mediapipe
* OS
* Numpy

## Project Setup
* Clone the project
* Open and RUN - **virtual_painter.py**
* Draw using Finger:
  * Selection Mode - Use Index and Middle fingers
  * Drawing Mode - Use Index finger
